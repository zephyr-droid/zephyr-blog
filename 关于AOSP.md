## 绕过强制门户检测
强制门户的作用是检测网络连通性，默认的服务器在国内无法使用。建议关闭这项设置，否则无线和数据网络信号会显示感叹号，还有可能自动断开网络连接
~~为什么不更换其他服务器？当然可以，但是如果你只用SIM卡打电话，数据网络信号上会显示感叹号，让人很不爽~~
```
adb shell settings put global captive_portal_mode 0
```

## 设置NTP服务器
用于同步时间
```
adb shell settings put global ntp_server pool.ntp.org
```

## 关闭专用DNS模式
设置，网络和互联网，专用DNS，选择专用DNS模式：关闭
